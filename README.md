# Comprehensive Setup Instructions

## Prerequisites
- Node.js (version x.x.x)
- NPM or Yarn
- Shopify account with access to the store

## How to Run the Store
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Gaurav-Sanap/lilsparkle-shopify-store.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd lilsparkle-shopify-store
   ```

3. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

4. **Run the development server:**
   ```bash
   npm start
   # or
   yarn start
   ```

5. **Access the store:**
   Open `http://localhost:3000` in your browser.

## Deployment Options
- **Deploy on Heroku:** 
   1. Create a new Heroku app.
   2. Connect the app to your GitHub repository.
   3. Enable automatic deploys from the `main` branch.

- **Deploy on Vercel:** 
   1. Sign in to Vercel.
   2. Import your GitHub repository.
   3. Configure your project settings and deploy.
